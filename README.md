## User Creation

### Objective

Create a cloud user account in Microsoft Entra ID to demonstrate identity lifecycle management and account provisioning.

### Steps Performed

1. Navigated to Microsoft Entra ID.
2. Opened Users > All Users.
3. Selected Create New User.
4. Configured a new user account named John Doe.
5. Generated a temporary password.
6. Created the user account.
